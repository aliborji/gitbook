# Detection

### Two Stage \(Region Proposal + Roi Head\)

* Faster RCNN
* Mask RCNN
* Feature Pyramid Network

### One Stage

* YOLOv1
* YOLOv2
* YOLOv3
* SSD
* CornerNet
* CenterNet

### Training

* Online Hard Example Mining \(OHEM\)

### Metrics

* Mean Average Precision \(mAP\)

### Datasets

* [COCO](http://cocodataset.org/)
